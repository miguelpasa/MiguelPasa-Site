Ideal Job

My ideal job is to be a Backend Developer. Here is a link to an ad that I found.

Included above is a screenshot of an ad for my ideal job. The job refers to building some RESTful Java Services that integrate with the company's front end services. Their architecture is monolithic, but the company wishes to modularise their solutions more in time, so this could also be part of the job. This is appealing to me firstly because I love Java. I also appreciate the challenge of a difficult solution, and I know that RESTful Java Services can be quite complex, perhaps implementing things like multithreading or even big data algorithms depending on the service needed.

Skills include experience in Java, Java server technologies (Spring, Spring Boot), the REST design (including REST API design), Java compilation and export technologies (Maven or Gradle) and experience with solutions for distributed systems. Other skills include infrastructure knowledge to do with Docker, AWS and FIX, and other backend technologies. The ad doesn't list any qualifications needed, however, I believe since it is a Mid to Senior level position, it would probably require a University degree in some Computer Science area, and also experience working in the IT industry.

I have extensive skills in Java, including REST APIs and designing solutions for the REST design. I also have experience using Java Spring and Spring Boot, and in using both Maven and Gradle and I have experience using JUnit to promote Test Driven Development. I feel I have enough experience to go into the job confidently with these things. However, I don't know how to develop solutions that would suit an architecture that is supported by distributed systems. I am also not very experienced with AWS, Docker, Kotlin or FIX protocol.

In order to improve these, I would like to take university courses that would give me more knowledge in backend service hosting and also gain more knowledge with Infrastructure - regarding the infrastructure of a network, and how best to identify the suitable design in the services that will be used for that network. Uni courses, web tutorials and youtube videos will help me get some of the way, but I will also have to experiment with this in my free time.
